# How to Run:
## run all tests:
    ./test.py

## run 1 test:
    ./test.py [test]

## list all tests
    ./test.py -l

# Add a Test
## add the following files:
    input/file/<test>.txt
    input/command/<test>.py
    output/file/expected/<test>.txt

## (optional) if you don't want to use input/vimrc/default.vim, add:
    input/vimrc/<test>.vim
